Правовая информация

MrDogez Project — программное обеспечение для оптимизации сетевого подключения на платформе Windows. Сервис предоставляет инструменты для улучшения качества интернет-соединения, обхода ограничений и защиты приватности.

Состав услуг

  • DPI Bypass — оптимизация маршрутизации трафика (27 стратегий: Fake TLS, multi-split, ECH-bypass, badack-increment и другие)
  • WARP Tunnel — защищённое подключение через сеть Cloudflare с автоматическим подбором endpoint
  • DNS-over-HTTPS — шифрование DNS-запросов для повышения приватности (20+ серверов)
  • Мониторинг сети — отслеживание качества подключения, пинг сервисов, статистика трафика
  • Автоматическое управление — watchdog, авто-переподключение, hot-swap прокси, ISP-кэширование
  • MITM-защита — автоматическое обнаружение и блокировка перехватывающих прокси
  • Premium-подписка — расширенные возможности (безлимитная скорость, 20 параллельных прокси, Авто-TUN)

Способ предоставления

Услуги предоставляются в электронном виде посредством программного обеспечения, устанавливаемого на компьютер пользователя. Доставка физических товаров не осуществляется. Доступ к платным функциям активируется автоматически после подтверждения оплаты платёжной системой.

Тарифные планы

  • Бесплатный — базовый DPI bypass, 5 ГБ трафика/мес, скорость до 1 МБ/с, 3 профиля, базовые DNS
  • Premium (199 ₽/30 дней) — безлимитный трафик и скорость, 20 прокси, Авто-TUN, гео-фильтр EU, все DNS

  • Буст скорости (99 ₽/30 дней) — снятие ограничения скорости без Premium

  • Операционная система: Windows 10 / 11 (x64)

Документы
Правовая информация

MrDogez Project — программное обеспечение для оптимизации сетевого подключения на платформе Windows.

Состав услуг

  • DPI Bypass — 27 стратегий оптимизации маршрутизации
  • WARP Tunnel — защищённое подключение через Cloudflare
  • DNS-over-HTTPS — шифрование DNS-запросов
  • Мониторинг — отслеживание качества подключения
  • Автоматическое управление — watchdog, авто-переподключение

Способ предоставления

Услуги предоставляются в электронном виде. Доступ активируется автоматически после оплаты. Системные требования: Windows 10/11 (x64), интернет.

Основания для возврата

  • Услуга не была предоставлена по техническим причинам
  • Двойное или ошибочное списание средств
  • Запрос в течение 24 часов, если услуга не активирована

Порядок

Заявка через @MrDogez в Telegram. Укажите email, дату, сумму. Рассмотрение — до 3 дней. Возврат тем же способом оплаты.

Собираемые данные

  • Email — регистрация и уведомления
  • Данные трафика — учёт тарифа
  • Данные карт не хранятся — обрабатываются платёжной системой

Пароли хешируются (PBKDF2-SHA256). Данные не передаются третьим лицам. Пользователь может запросить удаление.

Оплата услуг является акцептом оферты (ст. 437 ГК РФ). Исполнитель предоставляет доступ к функциям на условиях тарифа. Сервис предоставляется «как есть». Споры решаются путём переговоров в соответствии с законодательством РФ.
